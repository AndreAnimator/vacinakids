Projeto para desafio técnico da vaga de estagiário de Front End na Cyrrus.
O intuito dessa aplicação é substituir o uso de carteirinhas de vacinação para acompanhamento de vacinações infantis, mostrando novas campanhas, disponibilizando informações sobre vacinas, e se a carteirinha está atualiada com a criança tendo as vacinas em dia.

Para rodar, clone o repositório, tenha o Ionic e Firebase instalados em seu pc. Rode ```npm i``` e depois use ```ionic serve``` para rodar a aplicação. É possível que precise de uma extensão para habilitar o cors.
